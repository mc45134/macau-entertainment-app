# 澳门娱乐咨询 APP - 视觉设计方案 (v2.0)

## 产品定位

**产品名称**：澳门娱乐咨询

**核心价值**：融合澳门高端娱乐与葡式文化，为用户提供一站式AI智能娱乐资讯服务。

**意象锚点**：
> 步入澳门永利酒店大堂——深绿色大理石墙面在暖金色灯光下流转光泽，葡式拱廊与水晶吊灯交相辉映。空气中弥漫着优雅与奢华，每处细节都在诉说这座城市的独特气质。

---

## 设计风格

### 选定风格：澳门奢华绿金风 (Macau Luxe Green & Gold)

**气质与意象**：
- **关键词**：奢华、沉稳、温暖、异域、精致、典雅
- **场景描述**：深翠绿丝绒沙发在暖金色灯光下泛着柔和光泽，葡式蓝白瓷砖墙面上繁复花纹若隐若现，暖白大理石地面上映着吊灯光斑——信息如珍宝般陈列在精心设计的"展柜"卡片中。

### 配色方案

| 用途 | 色值 | 意象来源 |
|------|------|----------|
| **主色** | `#0A5C36` (深翠绿) | 澳门市花莲花之叶、永利/葡京的奢华绿装饰 |
| **暖金** | `#C9A96E` (奢华金) | 澳门赌场的金色装饰、澳门区徽的金色莲花 |
| **葡式蓝** | `#1A6B8A` (葡萄牙蓝) | 葡式 Azulejo 瓷砖的经典蓝色 |
| **陶土红** | `#C85D3E` (陶土红) | 葡式建筑屋顶瓦片、大三巴牌坊的赭石色 |
| **底色** | `#FCF8F3` (暖奶油白) | 葡式建筑的暖白墙壁 |
| **卡片底** | `#FFFFFF` (纯白) | 干净的信息承载面 |
| **深灰** | `#1A1A2E` (墨黑) | 深色文字，稳重 |
| **暖灰** | `#8B7355` (古铜灰) | 辅助文字，复古感 |
| **淡金** | `#F5F0E0` (香槟金底) | 卡片/标签浅金色背景 |

### 字体排版

- **英文数字**：Georgia / Playfair Display (衬线字体，奢华感)
- **中文**：系统自带苹方/思源宋体风格
- **排版策略**：
  - 标题：28-32px，Bold，字间距 1px
  - 正文字号：14-16px，行高 22-26px
  - 标签字号：11-12px，字间距 0.5px
  - 核心数字：使用加粗衬线字体突出

### 视觉策略

**摄影方向**：
- 卡片使用圆角阴影 + 暖金色光晕
- 图片下加金线装饰分隔
- 使用葡式花纹作为点缀装饰

**图形语言**：
- 卡片: rounded-2xl (20px), 暖金色阴影
- 按钮: 深绿底 + 金色文字, 按压 scale(0.96)
- 标签: 浅金底 + 深绿文字
- 分割线: 金色细线 1px

### 页面结构

**各模块差异化设计**：

| 模块 | 主色 | 装饰元素 |
|------|------|----------|
| AI咨询 | 深翠绿 + 暖金 | 魔法星星装饰 |
| 活动头条 | 葡萄牙蓝 + 白 | 葡式瓷砖花纹元素 |
| 电影 | 陶土红 + 金 | 胶片光影装饰 |
| 演唱会 | 深翠绿 + 玫瑰金 | 音符光晕装饰 |

### 动效与交互

- **点击反馈**：按钮 scale(0.96)，配合有色阴影压暗
- **卡片入场**：从下往上淡入 (translateY 8px → 0, opacity 0→1)
- **导航切换**：使用缓动曲线 easeInOut
- **加载状态**：金色旋转动画

### 设计禁忌

1. **禁止使用纯黑阴影** - 必须使用深绿色/金色暖调有色阴影 (rgba(10, 92, 54, 0.08))
2. **禁止使用 Emoji 作为图标** - 必须使用 FontAwesome6 矢量图标
3. **禁止卡片纯平面** - 必须用有色阴影 + 圆角层次
4. **禁止使用冷灰色文字** - 辅助文字使用暖灰 (#8B7355)
5. **禁止页面过于拥挤** - 保持 20px 以上间距，充足呼吸感
